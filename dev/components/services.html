How The Network Sustains Itself

Each node is a
revenue-generating
infrastructure asset.

A Fieldwork node is not a router in a box. It is a containerized data center — hardened, solar-powered, and built to generate commercial income around the clock. That income is what makes community-owned broadband sustainable without permanent grant dependence.

What lives inside each node

Broadband core

Fiber-fed, licensed microwave backhaul. Gigabit to the community.

Private LTE

CBRS spectrum. Community SIM cards. Coverage where no carrier has ever built.

Commercial compute

Enterprise compute infrastructure leased to AI and cloud clients around the clock.

NOAA weather

Certified monitoring stations feeding national weather and CAL FIRE systems.

LoRaWAN mesh

Emergency comms layer. Operates independent of power and internet.

Telehealth

Infrastructure for remote clinical services at anchor institutions.

The same fire station that hosts the container earns monthly lease income. The same towers that route your call lease space to Verizon. The same compute infrastructure that processes local AI services leases remaining capacity to enterprise clients worldwide.

No carrier built this because it didn't pencil out for them at commercial tower construction costs. It pencils out for Fieldwork because the infrastructure is grant-funded, the power is solar, and the returns stay in the community that owns it — not in a data center in San Jose.

The model scales

Every fire department and public safety facility that hosts a node adds capacity, coverage, and revenue. Each node is a franchise location for a network that grows because it pays to grow. Southern Humboldt is the proof of concept. The model works anywhere carriers have declined to build.
